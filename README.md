## üõ†Ô∏è Tech Stack

### **Core Architecture**
* **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly Typed)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Deployment:** [Vercel](https://vercel.com/)

### **Motion & UI**
* **Animations:** [GSAP (GreenSock)](https://gsap.com/) (ScrollTrigger, ScrollTo, Context API)
* **Data Viz:** [Recharts](https://recharts.org/) (Interactive Radar Charts)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Scrolling:** Custom Smooth Scroll implementation

### **Media Optimization**
* **Video Hosting:** [Cloudinary](https://cloudinary.com/) (Adaptive Bitrate Streaming)
* **Image Handling:** Next/Image with blur-up placeholders
